# Fix Smooth Translator - 穆译

> **特别感谢**  
> 本项目基于 [smooth-translator](https://github.com/greatghoul/smooth-translator) 升级重构而来。原项目使用 Chrome Manifest V2 架构（现已废弃），本项目沿用了原项目的图标/插件名和 toast 样式，并将代码重构为 Manifest V3 架构。

一款轻量高效的 Chrome 扩展，支持划词翻译和单词记录。

## 功能特性

- 🔍 **智能文本选择**：自动翻译选中的文本（支持单词和短语[3个词内]）
- 🔍 **链接划词查词**：快捷键 `Ctrl + Shift + L` 进入链接划词模式， 此时所有的链接的跳转会失效，你可以对其进行划词翻译，划词后，会自动退出链接划词模式。(可在设置内修改自定义快捷键）
- 💾 **单词记录**：自动保存翻译过的单词，支持设置不同保存期限
- ⚡ **快速响应**：优化的性能和超时控制
- 🎨 **简洁界面**：最小化、非侵入式的消息提示

## 翻译来源

- **有道词典**：用于词义与音标
- **有道翻译**：用于网络词汇翻译（至多3个单词）
- **金山词霸**：用于兜底翻译

## 安装方法

### 从源码安装

1. 克隆此仓库（或点击Code -> Download ZIP -> 解压到文件夹）：
```bash
git clone https://github.com/islder/FixSmoothTranslator.git
```

```bash
cd FixSmoothTranslator
```

2. 打开 Chrome 浏览器并访问 `chrome://extensions/`（其他浏览器可参考在 Extensions/扩展 内打开）

3. 在右上角启用「开发者模式」

4. 点击「加载已解压的扩展程序」并选择扩展目录

## 使用方法

1. **基本翻译**：只需选中网页上的任何文本，即可看到即时翻译

2. **设置选项**：点击扩展图标可访问：
   - 翻译自动消失时间设置
   - 单词记录选项
   - 导入/导出单词列表

## 单词记录功能

扩展可以自动保存您翻译过的单词：

- **保留期限**：可选择 15、30 、90 或 180 天
- **导出功能**：以 TXT 或 JSON 格式下载单词列表
- **导入功能**：从备份导入单词列表
- **自动清理**：根据保留设置自动删除过期单词

## 配置设置

访问选项页面：
- 右键点击扩展图标 → 「选项」
- 或点击弹出窗口中的设置按钮

## 隐私保护

- 不收集或传输任何个人数据
- 所有翻译使用公开 API
- 单词列表本地存储在您的浏览器中
- 无分析或跟踪
